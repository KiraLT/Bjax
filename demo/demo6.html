Bind Bjax events manually without jQuery selector

JavaScript (add to page HEAD): 

$(document).on('click', '[data-bjax]', function(event) {
    new Bjax(this);
    event.preventDefault();
});

Example HTML link: 

<a href="index.html">Click To Load Home</a>
