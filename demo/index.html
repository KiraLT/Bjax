Welcome to Bjax

Ajax link system for modern website. Convert any link in your page to ajax link.

Find more on GitHub

Bind Bjax on all links:

Javascript: 
Bjax.bindLinks('a')
HTML: 
<a href="/">Go to home</a>

Bind Bjax on data-bjax attribute:

Javascript: 
Bjax.bindLinks('[data-bjax]')
HTML: 
<a href="/" data-bjax>Go to home</a>

Call manually:

HTML: 
<a href="/" onclick="Bjax.loadLink(this.href); return false;">Go to home</a>
